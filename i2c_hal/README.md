# I2C HAL Implementation using Renode

## Overview
This project demonstrates a basic I2C HAL implementation for the NXP S32K388 platform using Renode.

## Features
- I2C HAL initialization
- START and STOP condition generation
- Byte transmission
- UART-based debug output
- Validation using Renode simulation

## Project Structure
- HAL/i2c : I2C HAL source and header files
- HAL/uart : UART HAL used for debug output
- src : I2C test application
- renode : Renode script for simulation
- Makefile : Build configuration
